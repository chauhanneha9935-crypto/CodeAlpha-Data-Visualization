# CodeAlpha-Data-Visualization

## 📌 Project Overview
This project focuses on transforming raw data into visual representations to better understand patterns and trends.

Data visualization helps in presenting insights clearly for decision-making.

## 🎯 Objectives
- Create visual representations of dataset
- Analyze distribution of tags
- Generate correlation heatmap
- Improve data storytelling

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- NumPy

## 📂 Project Files
task3_data_visualization.py  
quotes_dataset.csv  

## ▶️ How to Run

1. Install required libraries:
pip install pandas matplotlib seaborn numpy

2. Run the script:
python task3_data_visualization.py

## 📊 Visualizations Created
- Histogram (Tag Count Distribution)
- Correlation Heatmap

## 📈 Insights
- Tags vary across quotes
- Visualization helps identify distribution patterns
- Heatmap shows correlation between numeric variables

- Data visualization techniques
